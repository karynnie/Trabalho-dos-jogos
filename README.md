## 📌 Descrição do Projeto

Este projeto consiste em um **mini arcade desenvolvido em linguagem C**, contendo três jogos interativos executados via terminal. O objetivo é aplicar conceitos fundamentais de programação, como:

* Estruturas de repetição (`while`, `do-while`)
* Estruturas condicionais (`if`, `else`)
* Vetores (arrays)
* Funções
* Geração de números aleatórios (`rand`)
* Entrada e saída de dados (`scanf` e `printf`)

---

## 🎮 Jogos Disponíveis

### 1. ❓ Pergunta e Resposta

Um jogo de quiz com 5 perguntas de conhecimentos gerais.

**Funcionalidades:**

* Múltipla escolha
* Contagem de acertos
* Feedback imediato (correto/errado)

---

### 2. 🐍 Cobra na Caixa

Jogo de sorte entre dois jogadores.

**Como funciona:**

* Existem 5 caixas
* Uma contém uma cobra 🐍 (perde)
* Outra contém um botão 🟢 (vence)
* Os jogadores se alternam escolhendo caixas

**Recursos:**

* Escolha de nomes
* Sorteio de quem começa
* Sistema de repetição de partida

---

### 3. ⚔️ Goumas War

Jogo estratégico baseado em criaturas chamadas "Gousmas".

**Mecânicas principais:**

* Cada jogador começa com 2 Gousmas
* Cada Gousma possui um nível de **fúria**
* Ações disponíveis:

  * **Atacar:** aumenta a fúria do inimigo (pode destruí-lo)
  * **Dividir:** cria uma nova Gousma dividindo a fúria

**Regras importantes:**

* Se a fúria ultrapassar 5 → Gousma é destruída
* Jogador vence quando o adversário fica sem Gousmas
